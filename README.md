## 🎯 **Project Overview**

This repository contains a **web application** built using **PHP** and **JavaScript**, along with automated **Software Quality Testing (SQT)** scripts written in **Python** using **Selenium WebDriver**. The project focuses on validating the functionality, usability, and reliability of the web system through comprehensive automated end-to-end testing.

### **Why This Project?**
- ✅ Ensure web application reliability through automated testing
- ✅ Reduce manual testing effort and human error
- ✅ Catch regressions early in the development cycle
- ✅ Simulate real-world user interactions
- ✅ Maintain code quality and system stability

## ✨ **Features**

### **🌐 Web Application**
- Dynamic server-side functionality using PHP
- Interactive UI with vanilla JavaScript
- Responsive HTML/CSS design
- Form validation and submission handling
- Database integration (if applicable)
- Session management

### **🤖 Automated Testing**
- End-to-end UI testing using Selenium WebDriver
- Automated browser interaction (clicks, typing, navigation)
- Functional validation of key features
- Cross-browser testing support (Chrome, Firefox)
- Test report generation
- Screenshot capture on test failure
- Page Object Model (POM) design pattern
