Selenium-SQT-testing
This repository contains a web application built using PHP and JavaScript, along with automated software quality testing (SQT) scripts written in Python using Selenium WebDriver. The project focuses on validating the functionality, usability, and reliability of the web system through automated end-to-end testing.

🚀 Project Overview
The system includes a dynamic web application developed with PHP for backend logic and JavaScript for frontend interactivity. To ensure quality and stability, Selenium-based automated tests are implemented in Python to simulate real user interactions and verify expected outcomes.
